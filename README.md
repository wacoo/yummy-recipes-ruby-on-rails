# 📖 Yummy Recipe <a name="about-project"></a>



**Yummy Recipe** is a web application built with Ruby on Rails that simplifies the process of managing food recipes and tracking available ingredients. Users can create an account, allowing them to add, display, and archive their favorite recipes. The app also keeps track of the ingredients users have in their pantry. If a user lacks certain ingredients to prepare a recipe, the app automatically generates and displays a shopping list to ensure all necessary items are acquired. Yummy Recipe goes beyond individual user experiences by offering a public recipe feature, allowing users to share their culinary creations with a wider audience. With its intuitive interface and convenient features, Yummy Recipe empowers users to explore new dishes, manage their ingredients, and foster a vibrant cooking community.

## 💻 Getting Started <a name="getting-started"></a>

### Prerequisites

In order to run this project you need:
- Browser
- IDE
- PostgreSQL database installed
- Ruby on Rails
- Ruby


### Setup
Clone the repository to desired directory on local computer.
```
   git clone https://github.com/wacoo/yummy-recipes-ruby-on-rails

   cd yummy-recipes-ruby-on-rails
```

### Install
To install the project please run this command:
```
bundle install
```
### Usage
To start the project please run this commands:
```
rails db:create db:migrate
rails s
```
### Test
To run the tests please run this command:
```
rspec spec
```
